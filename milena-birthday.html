Как подключить общую стену
  1. Зайди на jsonbin.io → Sign Up (бесплатно)
  2. В меню нажми API Keys → скопируй Master Key
  3. Нажми New Bin → вставь [] → Create → скопируй BIN ID из адреса
  4. В этом HTML-файле найди первые две строки в <script> и замени значения
  5. Сохрани файл → загрузи на любой хостинг (например tiiny.host — перетащи файл, получи ссылку за 30 секунд)
